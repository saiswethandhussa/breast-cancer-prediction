# 🧬 Breast Cancer Prediction

## 📌 Overview

This project uses Machine Learning to predict whether a tumor is **Malignant (M)** or **Benign (B)** based on medical features.

## 🎯 Objective

To build a classification model that helps in early detection of breast cancer using data-driven techniques.

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib / Seaborn

## 🤖 Models Implemented

* Logistic Regression
* Decision Tree Classifier

## 📊 Results

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 97%      |
| Decision Tree       | 93.5%    |
